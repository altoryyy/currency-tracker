# 1 Введение

<a name="appointment"/>

## 1.1 Назначение
В этом документе описаны функциональные и нефункциональные требования к приложению «CurrencyTracker». Этот документ предназначен для команды, которая будет реализовывать и проверять корректность работы приложения.

<a name="business_requirements"/>

## 1.2 Бизнес-требования

<a name="initial_data"/>

### 1.2.1 Исходные данные
Курсы валют являются важным показателем для туристов, предпринимателей и частных лиц. Доступ к актуальной информации и её наглядной визуализации помогает принимать финансовые решения. Однако пользователи часто сталкиваются с необходимостью искать данные на разных сайтах или работать с неудобными сервисами.

<a name="business_opportunities"/>

### 1.2.2 Возможности бизнеса
Приложение «CurrencyTracker» позволит:  
- быстро получать актуальные данные о курсах валют;  
- просматривать историческую динамику в удобных графиках;  
- выбирать интересующие валюты и базу пересчёта;  
- предоставлять простой и понятный интерфейс для пользователей любого уровня подготовки.  

<a name="project_boundary"/>

### 1.2.3 Границы проекта
Приложение «CurrencyTracker» будет предоставлять пользователю:  
- актуальные данные о курсах валют;  
- просмотр графиков изменения курса за выбранный период;  
- возможность выбирать базовую валюту.  

Приложение **не будет**:  
- выполнять реальные финансовые операции (покупка/продажа валюты);  
- хранить платёжные данные пользователей.  

<a name="analogues"/>

## 1.3 Аналоги
Аналогами являются сервисы XE, Investing.com, Bloomberg. Их недостатки: перегруженность интерфейса, рекламные блоки, ограниченный бесплатный функционал. Преимуществом «CurrencyTracker» станет минималистичный дизайн, отсутствие рекламы и акцент на удобстве.

## 2.4 Предположения и зависимости
1. Приложение требует интернет-соединения для получения данных.  
2. Работа зависит от доступности выбранного API.  
3. Возможны ограничения бесплатного тарифа API (лимиты запросов).

# 3 Системные требования

<a name="functional_requirements"/>

## 3.1 Функциональные требования

<a name="main_functions"/>

### 3.1.1 Основные функции

<a name="view_exchange_rates"/>

#### 3.1.1.1 Просмотр актуальных курсов
**Описание.** Пользователь видит список популярных валют с курсом к базовой.  

<a name="select_base_currency"/>

#### 3.1.1.2 Выбор базовой валюты
**Описание.** Пользователь может переключить базовую валюту (например, USD → EUR).  

<a name="view_currency_charts"/>

#### 3.1.1.3 Просмотр графиков динамики
**Описание.** Пользователь может просматривать график курса валюты за 7, 30 или 90 дней.  

<a name="currency_search"/>

#### 3.1.1.4 Поиск валюты
**Описание.** Пользователь может найти валюту через строку поиска.  

<a name="update_data"/>

#### 3.1.1.5 Обновление данных
**Описание.** Система автоматически обновляет данные или позволяет обновить вручную.  

<a name="restrictions_and_exclusions"/>

### 3.1.2 Ограничения и исключения
1. Приложение не выполняет реальные финансовые операции.  
2. Для работы требуется интернет.  

<a name="non-functional_requirements"/>

## 3.2 Нефункциональные требования

<a name="quality_attributes"/>

### 3.2.1 Атрибуты качества

<a name="requirements_for_ease_of_use"/>

#### 3.2.1.1 Требования к удобству использования
1. Интерфейс должен быть интуитивно понятным.  
2. Все элементы должны быть подписаны.  
3. Минималистичный дизайн без лишних элементов.  

<a name="security_requirements"/>

#### 3.2.1.2 Требования к безопасности
1. Личные настройки доступны только зарегистрированному пользователю.  
2. Приложение не хранит платёжные данные.  

<a name="access_requirements"/>

#### 3.2.1.3 Требования к доступности
1. Время отклика системы не более 2 секунд.  

<a name="external_interfaces"/>

### 3.2.2 Внешние интерфейсы
1. Интерфейс соответствует современным стандартам веб-дизайна (responsive).  
2. Поддержка работы на мобильных устройствах и ПК.  

<a name="restrictions"/>

### 3.2.3 Ограничения
1. Приложение реализовано как веб-приложение.  
2. Язык разработки — JavaScript (React + Node.js).  
3. Язык интерфейса — русский / английский.
